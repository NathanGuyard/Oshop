# oShop

Oshop et un back-office fait pendant ma formation, il permet de gérer des catégories tag ajout d'utilisateur supprimer et modifier.

## Technologies utilisées

- PHP 7.2
- POO
- MVC
- AltoRouter
- Composer
- Symfony (Var_dump)
- Alto-dispatcher
- SQL
- BDD
- Adminer
